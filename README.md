# Asset-tracker

※해당 확장은 국내 사이트 '깡통 사이트'에서 만들어지고 배포되었습니다.

※해당 확장은 '깡통 사이트'에서 배포된 character-assets 확장을 필수적으로 필요로 합니다(깃허브 링크:https://github.com/tincansimagine/character-assets).

📊 SillyTavern Asset Tracker
(실리태번 애셋 트래커)
SillyTavern에서 캐릭터와의 대화 중 등장하는 스프라이트(이미지)나 애셋을 자동으로 감지하여 수집률(%)을 추적하고, 게임의 도전과제(Achievement)와 같은 경험을 제공하는 확장 기능입니다.

✨ 주요 기능 (Features)
📊 실시간 수집 현황 추적: 캐릭터의 애셋 파일이 채팅에 출력되면 자동으로 '해금(Unlock)' 처리됩니다(같은 캐릭터 카드를 공유한다면 다른 채팅방의 애셋 달성률과 통일됩니다).

🏆 100% 달성 히든 메시지: 모든 애셋을 모으면 캐릭터 제작자가 숨겨둔(또는 유저가 설정한) 축하 메시지가 팡파레와 함께 출력됩니다.

🔔 발견 알림 (Toast): 새로운 애셋을 찾을 때마다 실시간으로 알림을 띄워줍니다. (On/Off 가능)

😈 치트 모드 (Cheat Mode): 잘 안 나오는 애셋은 리스트를 클릭하여 강제로 해금할 수 있습니다. (치트 사용 시 ☑️ 아이콘으로 구분됨)

🔄 자동 동기화: 서버의 파일 변경 사항(추가/삭제)을 실시간으로 감지하여 리스트에 반영합니다.

🛡️ 스포일러 방지: 히든 메시지는 기본적으로 가려져 있으며, 토글을 통해서만 확인 가능합니다.

📖 사용 가이드 (User Guide)
1. 기본 사용법
실리태번의 확장 기능(Extensions) 메뉴에서 Asset Tracker 패널을 엽니다.

**"이 채팅방에서 추적 활성화"**를 체크하면 현재 캐릭터의 애셋 목록을 불러옵니다.

캐릭터와 대화하며 이미지가 출력되면, 자물쇠(🔒)가 풀리고 체크 표시(✅)로 바뀝니다.

2. 치트 및 취소
강제 해금: 잠겨있는 항목(🔒)을 클릭하면 치트 모드로 해금할 수 있습니다. (☑️ 표시)

수집 취소: 이미 해금된 항목(✅, ☑️)을 클릭하면 수집 기록을 삭제하고 다시 잠금 상태로 되돌립니다.

(수집의 강제 해금과 수집 취소는 추적 활성화 토글이 비활성화 되어있으면 불가능합니다.)

3. 알림 설정
🔔 새 애셋 발견 시 알림: 켜두면 새로운 이미지가 나올 때마다 3초간 토스트 메시지가 뜹니다.

🎨 캐릭터 제작자 가이드 (For Character Creators)
캐릭터 제작자는 카드를 배포할 때, 100% 달성 시 유저에게 보여줄 히든 메시지를 카드 안에 심을 수 있습니다.

📝 작성 방법
캐릭터 카드의 Character's Note (또는 Author's Note, Depth Prompt) 란에 아래 코드를 적으세요.

Ex. [System: asset_complete='축하합니다! 모든 기억을 되찾으셨군요!']

형식: asset_complete='메시지 내용'

주의: AI가 이 코드를 대화 문맥으로 인식하지 못하도록, 반드시 대괄호 [ ] 나 HTML 주석 `` 등으로 감싸거나, 빈도(Frequency)를 0으로 설정하는 것을 권장합니다.

Character's Note 추천: 실리태번의 [Character Editor] -> [Character's Note] 란에 적으면, 유저의 캐릭터 리스트에는 보이지 않으면서 카드 파일(PNG)에는 안전하게 저장되어 배포됩니다.
